## Description

A JavaScript application based on the Odin Project exercise - **'Etch-a-Sketch'**

**Features:**

- Modern User Interface
- Keyboard Support
- Duo-Pen (+ Mouse Support)

## What I've Learnt

+ Advanced DOM manipulation
+ Handling mouse events
+ Using data-attributes to store data
