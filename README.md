## Screenshots

![Showcasing different brush modes](/assets/images/brush-modes.png)

## Description

Draw anyhting using this sketchpad, and mostly importantly, have fun!

**Features:**

- Modern User Interface
- Simple Brush Modes
- Customizable Sketchpad
- Keyboard Support
- **NEW**: Duo-Pen (+ Mouse Support)
